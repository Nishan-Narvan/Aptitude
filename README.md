# TCS NQT Numerical Ability – Quick Revision Guide

A compact checklist of the **most common aptitude topics, formulas, and solving templates** for TCS NQT.

---

# 1. Percentages

### Core Formula
```
x% of y = (x/100) × y
```

### Percentage Increase / Decrease
```
Increase % = (Increase / Original) × 100
Decrease % = (Decrease / Original) × 100
```

### Net Percentage Change
```
Net % ≈ a + b + (ab/100)
```

Example:
```
Increase 20% then decrease 10%
Net change ≈ 20 - 10 - 2 = 8%
```

### Find Original Value
```
Original = Final / (1 + % change)
```

---

# 2. Ratio & Proportion

### Basic Rule
```
a : b = c : d
ad = bc
```

### Divide Quantity in Ratio
```
Value = Total × Part / (Sum of Ratio Parts)
```

Example:
```
Divide ₹840 in ratio 3:4

Total parts = 7

3/7 × 840 = 360
4/7 × 840 = 480
```

---

# 3. Averages

### Core Formula
```
Average = Total / Count
Total = Average × Count
```

---

### Missing Number
Steps:
```
1. Total = avg × count
2. Sum known numbers
3. Missing = total − known sum
```

---

### Replacement

```
Joined − Left = (New Avg − Old Avg) × Number of items
```

---

### Combined Average

```
Combined Avg = (A1N1 + A2N2) / (N1 + N2)
```

Where
```
A1 = avg group1
N1 = number group1
```

---

### Alligation (Weighted Average)

```
Ratio = (A2 − A) : (A − A1)
```

Example:
```
Adults avg = 15
Children avg = 7
Family avg = 12

Adults : Children
= (15-12):(12-7)
= 3:5
```

---

# 4. Profit & Loss

### Core Formulas
```
Profit = SP − CP
Loss = CP − SP
```

### Percent
```
Profit % = (Profit / CP) × 100
Loss % = (Loss / CP) × 100
```

---

### Successive Discount
```
Net discount = a + b − (ab/100)
```

Example:
```
10% + 20% discount

Net discount
= 10 + 20 − 2
= 28%
```

---

# 5. Simple Interest

```
SI = (P × R × T) / 100
```

Where
```
P = Principal
R = Rate
T = Time
```

---

# 6. Compound Interest

```
Amount = P(1 + r/100)^t
CI = Amount − Principal
```

---

# 7. Time, Speed & Distance

### Core Formula
```
Speed = Distance / Time
Distance = Speed × Time
Time = Distance / Speed
```

---

### Relative Speed

Opposite direction
```
S1 + S2
```

Same direction
```
|S1 − S2|
```

---

### Boats and Streams

```
Downstream = u + v
Upstream = u − v
```

Where
```
u = speed in still water
v = stream speed
```

---

# 8. Time & Work

### Work Formula
```
Work = Rate × Time
```

If A finishes work in x days

```
Rate of A = 1/x
```

Two people working together

```
1/x + 1/y
```

---

# 9. Number System

### LCM / HCF

```
LCM × HCF = Product of numbers
```

---

### Divisibility

Common rules:

```
Divisible by 3 → sum of digits divisible by 3
Divisible by 9 → sum of digits divisible by 9
Divisible by 11 → difference of alternating digits
```

---

# 10. Probability

```
Probability = Favourable outcomes / Total outcomes
```

# Quick Problem Solving Checklist

When reading any aptitude question ask:

```
1. Is this a percentage change?
2. Is this a ratio division?
3. Is this an average change?
4. Is someone replaced or added?
5. Is this work-rate or speed-distance?
```

Identify the **pattern first**, then apply the template.
